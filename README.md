**Simple CRM Dashboard**

A simple and responsive CRM Dashboard built to manage client leads efficiently. This project helps businesses track leads, update statuses, and manage follow-ups in an organized way.

**Purpose of the Project**

This project was developed as Task 2 – Client Lead Management System (Mini CRM) for the Full Stack Web Development Internship by Future Interns.

The main objective of this project is to understand:

• CRUD Operations
• Backend API Integration
• Database Management
• Business Workflow Handling

**Features**
• Add new client leads
• View all leads in dashboard
• Update lead status
• Delete leads
• Lead details with notes and follow-ups
• Responsive user interface
• Simple and clean dashboard design
• Lead Status Types
New
Contacted
Converted
