**Simple CRM Dashboard**

A simple and responsive CRM Dashboard built to manage client leads efficiently. This project helps businesses track leads, update statuses, and manage follow-ups in an organized way.

**Purpose of the Project**

This project was developed as Task 2 – Client Lead Management System (Mini CRM) for the Full Stack Web Development Internship by Future Interns.

The main objective of this project is to understand:

• CRUD Operations
• Backend API Integration
• Database Management
• Business Workflow Handling

**Features**
• Add new client leads
• View all leads in dashboard
• Update lead status
• Delete leads
• Lead details with notes and follow-ups
• Responsive user interface
• Simple and clean dashboard design
• Lead Status Types
New
Contacted
Converted

**Setup Instructions**

**1. Clone the Repository**
git clone https://github.com/Logicwithaditya/FUTURE_FS_02.git

**2. Open Project Folder**
cd FUTURE_FS_02

**3. Install Dependencies**
npm install

**4. Configure Database**

Create a MySQL database and update your database configuration with:
host: "localhost"
user: "root"
password: "your_password"
database: "leadflow_crm"

**5. Start the Server**
node server.js

**6. Open the Project**

**• Frontend:**
http://localhost:3000

**• Backend:**
http://localhost:5000
